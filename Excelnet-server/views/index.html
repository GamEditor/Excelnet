What is Excelnet Server?

Excelnet is a dynamic and run-time file-generator for client side requests. In this free and open source version of Excelnet server, you can get relative download links and you should handle correct download address yourself.

How to get download link from Excelnet API

It is so simple! Just make an ajax/web request from your client and send your data in http request body. for example look at the example-site within the repository.
